# Ansible Role: HHVM

Installs facebook's HHVM on CentOS 7 servers.

## Requirements

None.

## Role Variables

Ansible variables used in the role and their default values (see defauls/main.yml):

```yaml
hhvm_version: 3.5.0
```

Currently available versions are: ```3.3.3```, ```3.4.2``` and ```3.5.0```.
